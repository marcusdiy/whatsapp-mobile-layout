# whatsapp-mobile-layout
Lets make whatsapp mobile again! (on desktop, that's it)

WhatsApp Web feels like it’s frozen in time.
It demands half your screen like it’s 2015. Try using it alongside code, docs, design tools, or even a browser tab split — it just breaks down. 
The layout collapses, text gets awkward, and it’s clearly not designed to be resized down a smaller layout.

In a world where multitasking is the default, WhatsApp Web insists on being the main character.

After getting annoyed one too many times, I finally thought: fine, I’ll try to fix it myself.

Here my attempt!
